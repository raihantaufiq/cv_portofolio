Panji Sang Penjelajah: Game Edukasi Berpikir Komputasional

Game Edukasi yang dibuat sebagai produk topik skripsi. Penelitian lebih berfokus pada metode pengembangan agar menghasilkan game yang memberikan pengalaman bermain (game experience) yang baik. Target pemain dari game ini adalah siswa SMP/sederajat. Siswa diminta untuk mencoba menyelesaikan level dalam game, setelah itu mengisi kuesioner mengenai pengalaman bermain yang dirasakan.

Game Panji sang Penjelajah merupakan yang didesain agar pemain menyelesaikan permasalahan menggunakan prinsip berpikir komputasional. Game dikembangkan menggunakan C# dan Godot Game Engine. Metode pengembangan game menggunakan GDLC dan Framework MDA. Game dapat dimainkan pada platform Windows dan Android. Dokumentasi Teknis juga disusun untuk memaparkan struktur dan sistem game secara komprehensif.
