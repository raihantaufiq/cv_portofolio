Website Online Course

Website ini merupakan proyek yang dibuat untuk memenuhi tugas kuliah. Website dikembangkan CodeIgniter4 sebagai framework PHP dan Bootstrap sebagai framework Front-end. Arsitektur pemrograman menggunakan Model-View-Controller

Website memiliki 2 tampilan, yaitu tampilan untuk pengguna dan tampilan untuk admin. Sisi pelanggan berfungsi untuk melihat info course, pendaftaran course, dan megirim pesan. Sisi admin berfungsi untuk mengelola daftar course dan melihat pesan yang dikirim user.
