Dashboard Data Kesehatan Balita Desa Sariwangi

Dashboard ini dibuat ketika kegiatan KKN di Desa Sariwangi Bandung. Dashboard menampilkan data SKDN balita yang datang ke Posyandu untuk diperiksa tiap bulan. Data dikumpulkan menggunakan Google From, data tersebut akan secara otomatis masuk ke Google Sheets. Data pada Google Sheets kemudian dihubungkan dengan Looker Studio untuk dibuatkan visualisasi datanya. Data yang ditampilkan dashboard dapat diupdate ketika ada data baru yang masuk. Pengguna bisa menentukan pengaturan ketika ingin melihat data pada periode atau posyandu tertentu.

Keterangan: Data yang ditampilkan pada web ini adalah data dummy (bukan data asli)
